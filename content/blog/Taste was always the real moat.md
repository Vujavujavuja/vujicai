Why does it seem like everything is code now? Because generation got cheap.

![Quote card reading: "The big lesson from AI is that everything is code. A slide deck is code. Design is code. That cool promo video? Code. Excel automation? Code. The universe? Probably made of code too." Attributed to Guillermo Rauch (@rauchg), via X.](/thoughts/taste-was-always-the-real-moat/everything-is-code-rauch.webp)

Slide decks are just HTML with some CSS placed nicely, promo videos can be fully done using HTML and a bit of Remotion, and promo images turn out better when you ask your agent to do them using HTML instead of using an image model.

I have been playing around with these systems for some time now, and I have to say, on the surface it works really well and really fast. However, looking a bit closer you start noticing mistakes. Generating 100 videos is quick. Reviewing each of them is an endless task.

Which means the work was never the generation itself. It was coming up with the ideas, and writing the guidelines tightly enough that you are not just eyeballing the output.

So the job did not go away, it moved. Away from the making, which anyone can now do in an afternoon, and toward the deciding. What is worth producing, what does good look like, and would you actually put your name on this one. That part has no shortcut yet.

Turns out taste was always the real moat.
